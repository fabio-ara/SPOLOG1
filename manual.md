# Manual do Repositório SPOLOG1

## 1. Finalidade

Este repositório existe para organizar os exercícios e materiais da disciplina de Lógica de Programação I do IFSP.

Ele deve permanecer separado dos projetos pessoais e dos repositórios de outras disciplinas, para facilitar consulta, manutenção e apresentação acadêmica.

## 2. Estado atual

No ciclo atual, o repositório foi criado com documentação inicial, mas a estrutura interna dos exercícios ainda será definida em etapa posterior.

Enquanto essa definição não acontece, a raiz deve permanecer simples e conter apenas a documentação essencial do repositório.

## 3. Direção de organização

Quando os exercícios forem trazidos para este repositório, a organização deverá seguir a lógica real da disciplina.

Prioridade de decisão:

1. respeitar a forma como a disciplina distribui o conteúdo;
2. preservar a ordem pedagógica do semestre;
3. manter nomes de pastas e arquivos claros e consistentes.

## 4. Descrição sugerida no GitHub

Sugestão para o campo `About` do repositório:

`Repositório da disciplina de Lógica de Programação I do IFSP São Paulo.`

## 5. Regra de manutenção

- toda mudança estrutural relevante deve atualizar este `manual.md`;
- o `README.md` deve continuar servindo como apresentação inicial do repositório;
- quando a estrutura dos exercícios for definida, ela deve ser documentada no mesmo ciclo de trabalho.
